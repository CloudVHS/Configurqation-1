## 1. Команды для сборки и запуска проекта

### Установка необходимых библиотек:

   ```bash
   ppip install -r requirements.txt (requests, beautifulsoup4)
   ```

### Запуск проекта

   ```bash
   python visualizer.py <название пакета> <глубина>
   ```
#### Пример:

   ```bash
   python visualizer.py iptables 2
   ```

### Проверка

   ```bash
    python ./tests.py
   ```
